# advent-calendar
This site is a simple advent calendar site! You can log in and create unlimited free advent calendars that you can share with your friends & family. Behind every door, you can put messages, links to videos & pictures and more!

## Features
- Create advent calendars
- Share them with friends & family with a URL
- Password-protect your advent calendar (ex: you want to put in a gift card code)
- Users cannot open doors in the future, they can open past doors though!

## Tech Stack
The site relies on these main components:
- Next.JS as a web framework
- PostgreSQL for the database
- NeonAuth for authentification (also supports GitHub & Google!)
- Argon2 for password hashing
